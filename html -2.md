# html -2

## html标签

```html
<h1>标题标签<h1>
```

标题标签：作为标题使用，并且依据重要性递减

特点： 

1. 加了标题标签的文字会变加粗，字号也会依次变大
2. 标题标签会占据一行

+++

```html
<p>段落标签</p>
```

段落标签：作为段落使用

特点：

1. 文本在一个段落中会根据浏览器的窗口大小自动换行
2. 段落和段落之间会存在空隙

+++

```html
<br/>
```

换行标签

特点：

1. 单标签
2. br标签只是简单的换行，跟段落不一样，段落之间会插入一些垂直的间距

+++

文本格式化标签

1. 加粗 

```html
<strong></strong>
<b></b>
```

推荐使用 strong ，语义更强烈

2. 倾斜

```html
<em></em>
<i></i>
```

推荐使用em ，语义更强烈

3. 删除线

```html
<del></del>
<s></s>
```

推荐使用del， 语义更强烈

4. 下划线

```html
<ins></ins>
<u></u>
```

推荐使用ins ，语义更强烈

+++
